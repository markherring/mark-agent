# Standard Interview Answers

This folder contains your prepared responses to common interview questions, formatted using the STAR methodology (Situation, Task, Action, Result).

## How to Add New Answers

1. Copy `_template.md` to a new file (e.g., `technical_challenge.md`)
2. Fill in the question variants at the top
3. Write your answer using STAR format
4. Add key takeaways at the bottom

## Suggested Topics to Add

- Technical challenge you've solved
- Leadership style / team management
- Database architecture decision
- Conflict resolution
- Biggest professional achievement
- Biggest failure / what you learned
- Why you're looking for a new role
- Questions you have for the interviewer
- 30/60/90 day plan approach
